### Required
* `vim` should be installed.

### Features
* Vundle installed
* Lightline Installed
* AutoPairs Installed
* AutoComplPop Installed
* Everforest theme Installed
* Syntax Highlighting Enabled

### Lack Of

### Installation

```shell
wget shorturl.at/pMUW9
tar -xvf theme-for-vim.tar.gz
mv vim ~/.vim
mv vimrc ~/.vimrc
```
